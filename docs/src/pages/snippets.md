# Snippets

Snippets are reusable text blocks you can save and recall.

## Creating snippets

**From settings:**

1. Open settings (gear icon)
2. Scroll to the Snippets section
3. Type in the "Add snippet" field and press Enter

**From terminal output:**

1. Select text in the terminal output
2. A "Save snippet" button appears at the bottom
3. Tap it to save the selected text as a snippet

## Editing snippets

Tap a snippet in the settings list to open the edit modal. Modify the
text and tap Save.

## Deleting snippets

Tap the **x** button next to a snippet in the settings list.

## Using snippets

**From the command palette:**
Snippets appear in the command palette under the "Snippets" section.
Selecting one fills the text input with its content.

**From the snippets button:**
In terminal mode (when Claude Code is not active), the snippets button
opens the command palette filtered to snippets only.

## Storage

Snippets are stored in your browser's local storage. They persist across
sessions but are specific to the browser and device you saved them on.
