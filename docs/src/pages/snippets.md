# Snippets

Reusable text blocks you can save and recall.

## Creating

**From settings:** open settings, scroll to Snippets, type in the "Add
snippet" field and press Enter.

**From terminal output:** select text in the output — a "Save snippet"
button appears at the bottom.

## Editing

Tap a snippet in the settings list to open the edit modal.

## Deleting

Tap the **x** next to a snippet in settings.

## Using

**Command palette:** snippets appear under the Snippets section.
Selecting one fills the text input.

**Snippets button:** in terminal mode, tapping the snippets button opens
the palette filtered to snippets.

## Storage

Snippets live in your browser's local storage — they persist across
sessions but are specific to each browser and device.
