Wormhole

Terminal telekinesis

Setup

1

Install

Requires Node.js and tmux.

curl -sL https://raw.githubusercontent.com/cszach/wormhole/main/install.sh | sh
2

Run

npm run dev

Starts on port 5173 by default. Configure in .env.

3

Connect

Enter your server's address above. Use Tailscale for secure access from your phone.

4

Voice (optional)

Voice dictation requires HTTPS. With Tailscale, run tailscale cert and add paths to .env:

TLS_CERT=/path/to/cert.pem TLS_KEY=/path/to/key.pem
