# Themes & Customization

## Animated backgrounds

| Theme        | Description           |
| ------------ | --------------------- |
| Starry Night | Twinkling star field  |
| Aurora       | Northern lights waves |
| Nebula       | Cosmic gas clouds     |
| Topography   | Contour map lines     |

Select a theme in Settings > Themes.

## Accent colors

Eight accent colors tint the entire UI — buttons, active states,
borders, and highlights:

Lavender, Cyan, Emerald, Rose, Amber, Coral, Teal, White.

Your choice is saved in local storage and carries over to the docs pages.

## Terminal columns

- **Auto** (default) — calculates columns from your screen width
- **Manual** — set a specific count with the slider (40–200)

The column width is sent to tmux when you connect or change the setting.

## Install as app

Wormhole is a Progressive Web App. Install it for a full-screen
experience without browser chrome.

**iOS Safari:** Share > Add to Home Screen

**Android Chrome:** Menu > Add to Home Screen
